# Resume Workflow

Use this when starting a new session on this project.

## Steps

1. **Read the concept**
   - Open and read `CONCEPT.md` to understand what this project is about
   - This is a novel AI training paradigm combining debate + token economy

2. **Check current state**
   - Read the TOP entry in `DEVLOG.md` (most recent session)
   - Note: Open Questions and Next Steps

3. **Check architecture**
   - Read `docs/architecture.md` for system design decisions

4. **Ask the user**
   - "I've reviewed the project. Last session we [X]. What would you like to work on today?"

## Key Context

- **Goal**: Optimize internal AI reasoning (ITMC) via economic pressure
- **Hardware**: 8GB VRAM, 38GB RAM, Intel Ultra 9 + NPU
- **Models**: Targeting 7B quantized (4-bit) or 1-3B models
- **Status**: Foundation phase - documentation and scaffolding
